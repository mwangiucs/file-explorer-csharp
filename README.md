# FileExplorerCS (WPF)

Windows-style file explorer built with C# and WPF.

## Features

- Drive/folder tree (left pane)
- File and folder list (right pane)
- Path bar (type a path and press Enter)
- Back, Forward, and Up navigation
- Real filesystem browsing using `System.IO`

## Requirements

- Windows
- .NET 8 SDK

## Run

From this folder:

```powershell
dotnet restore
dotnet run
```

If `dotnet` is not recognized, install the SDK from:

https://dotnet.microsoft.com/download/dotnet/8.0
